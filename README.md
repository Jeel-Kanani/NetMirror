# NetMirror 🎬

> A social entertainment platform to discover, track, discuss, and share movies, series, anime, and games.

## Overview

NetMirror is a long-term full-stack project being built to explore modern backend and frontend development using Spring Boot and React.

The goal is to create a platform where users can:

* Discover movies, TV series, anime, and games
* Create personal watchlists
* Track watched content
* Get recommendations from friends
* Join discussion groups
* Share reviews and ratings
* Explore entertainment trends
* Find where content is available to watch
* Build entertainment communities

The project is currently in active development and is being built incrementally while learning industry-standard software development practices.

---

## Vision

Most entertainment platforms focus on only one aspect:

* IMDb → Movie information
* Letterboxd → Movie tracking
* Netflix → Streaming
* Reddit → Discussions

NetMirror aims to combine these experiences into a single platform.

Users should be able to:

* Save content for later
* Maintain watch history
* Discover new content
* Follow friends
* Join communities
* Share recommendations
* Discuss movies and series
* Build personalized entertainment profiles
